# 📊 Employee Performance Analysis using Power BI

 📌 Overview
This project focuses on analyzing employee performance data using Power BI. The dashboard provides meaningful insights into employee productivity, performance ratings, and overall efficiency to support data-driven decision-making.

 🎯 Objective
The main objective of this project is to:
- Monitor employee performance
- Identify top performers
- Analyze departmental efficiency
- Highlight areas for improvement

⚙️ Tools & Technologies
- Power BI
- Excel / CSV Dataset
- Data Cleaning & Transformation
- Data Visualization

 📊 Features
- Interactive dashboard with filters and slicers
- Department-wise performance analysis
- KPI indicators (Top performers, average ratings, etc.)
- Attendance and productivity tracking
- Visual reports using charts and graphs

📈 Key Insights
- Identified high-performing employees and departments
- Detected performance gaps
- Provided clear insights for management decision-making

 📷 Dashboard Preview



🚀 How to Use
1. Download the `.pbix` file from this repository  
2. Open it in Power BI Desktop  
3. Explore the dashboard using filters  

📌 Conclusion
This project demonstrates how Power BI can be used to transform raw employee data into actionable insights and improve organizational performance.
